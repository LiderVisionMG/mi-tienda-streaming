# 🎬 Streaming Store

Bienvenido a **Streaming Store**, una tienda virtual donde puedes adquirir tus servicios de streaming favoritos de manera fácil y segura.

## 📂 Estructura del proyecto

- `index.html` → Página principal con:
  - Banner dinámico (carousel).
  - Descripción general de la tienda.
  - Ilustraciones de lo que ofrecemos.
  - Footer con enlaces a redes sociales.

- `productos.html` → Página de productos con:
  - Opciones de plataformas (Netflix, Disney+, HBO Max, Spotify).
  - Selección de tipo de cuenta (compartida o completa).
  - Carrito que recopila las selecciones y redirige a WhatsApp.

## 🚀 Cómo navegar

1. Abre `index.html` para entrar al sitio.
2. Desde el menú superior, haz clic en **Productos** para ir a la página de compra.
3. Selecciona las plataformas y el tipo de cuenta que deseas.
4. Haz clic en el botón **Carrito** para enviar tu pedido directamente por WhatsApp.

## 📱 Integración con WhatsApp

El carrito genera un mensaje automático con tus selecciones y abre un chat en WhatsApp con el número configurado.  
> **Nota:** Cambia el número de WhatsApp en el código (`productos.html`) por el tuyo en formato internacional.  
Ejemplo: `573001234567` (57 = código de Colombia).
